# Ctrl+Alt+Delete

A comprehensive platform consisting of mobile, backend, and UI components.

## Project Structure

This project is organized into three main components:

- **seva_mobile/** - Mobile application
- **seva_platform/** - Backend platform services
- **seva_ui/** - User interface components

## Getting Started

### Prerequisites

- Node.js (for UI components)
- Mobile development environment (for mobile app)
- Backend runtime environment (for platform services)

### Installation

1. Clone this repository
2. Navigate to each component directory and follow their respective setup instructions

## Development

Each component can be developed independently:

- `cd seva_mobile` - Work on the mobile application
- `cd seva_platform` - Develop backend services
- `cd seva_ui` - Build UI components
